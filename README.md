# customer-api
A sample Platform Engineering project using RAML.

The project covers the scenario of having a restful API that contains a single resource, customers, and allows the following:
•   List customers
•   Create a new customer
•   Update a customer
•   Deletes a customer



## Getting Started

Please clone and/or download the project and keep the folder structure intact.


### Prerequisites

This project was developed using these optional but helpful tools:

- NodeJS (https://nodejs.org/en/)
- API Designer (https://www.mulesoft.com/platform/api/anypoint-designer)


### Installing


1. install and update NodeJS. (https://docs.npmjs.com/getting-started/installing-node)
2. install Api Designer. https://github.com/mulesoft/api-designer
3. run Api Designer

```
   $ api-designer
```

4. a new browser window will open and Api Designer will load.
5. click on Project > Import to load the project.
